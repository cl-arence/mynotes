# MindPad

MindPad is a Flutter notes app built from the freeCodeCamp Flutter course and then modernized into a darker, more polished product-style reference app.

## Highlights

- Firebase Authentication with login, register, forgot-password, email verification, and BLoC-driven auth flow.
- Firestore-backed notes with real-time syncing.
- Dark MindPad visual theme, app icon branding, searchable notes, improved empty states, and a warmer editor.
- Email verification reminders that let users keep using the app while they verify.
- Android-first release workflow from Windows, with iOS release deferred to cloud Mac tooling when needed.
